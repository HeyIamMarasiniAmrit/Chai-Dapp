# ☕ Chai DApp

A full-stack decentralized application (DApp) where users can connect their Ethereum wallet, send ETH, and leave messages stored permanently on the blockchain.

## 🚀 Live Demo

Add your Netlify link here:

https://chaiorder.netlify.app/


## 📌 About The Project

Chai DApp is a Web3 application built to understand how frontend applications interact with Ethereum smart contracts.

The project covers the complete DApp development workflow:

- Writing Solidity smart contracts
- Deploying contracts using Hardhat
- Connecting frontend with blockchain
- Interacting with MetaMask wallets
- Sending blockchain transactions
- Reading blockchain data

## ✨ Features

- 🔐 Connect Ethereum Wallet
- 💸 Send ETH transactions

- 📝 Store messages on blockchain
- 📖 Read blockchain-stored messages
- 🔗 Smart Contract Integration
